# 一、工业4.0和IIoT
无缝实时通信是所有工业4.0和工业物联网(lloT)实现的核心先决条件。瑞萨电子为所有领先的工业通信协议提供一流的通信组件，如PROFINET，以太网/IP, EtherCAT, Modbus, CC-Link等。所有这些硅解决方案都配有全面的软件支持。
